# Luxury Portfolio — React + Tailwind

Premium, modern personal portfolio scaffold built with React, Tailwind CSS, and Framer Motion.

Palette:
- Primary: #0F172A (Deep Navy Blue)
- Secondary: #1E3A8A (Royal Navy)
- Accent Glow: #38BDF8 (Soft Sky Blue)
- Luxury Highlight: #60A5FA (Premium Blue)
- Background Light: #F8FAFC
- Background Dark: #020617

Quick start

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Notes

- Theme toggle uses the `dark` class on the document element.
- Replace the placeholder portrait on the Home page with a high-resolution photo.
- Add `resume.pdf` to the project root to enable resume download.
